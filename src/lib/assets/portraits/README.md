# Portraits

Drop author photos here (jpg / png / heic / webp — any size, ideally ~3:4 and
face-centered). `@sveltejs/enhanced-img` optimizes them at build time (avif/webp

- responsive sizes), and the About page rotates through whatever is in this
  folder on each visit. With no images present, the About page shows the
  diagonal-hatch placeholder instead.
